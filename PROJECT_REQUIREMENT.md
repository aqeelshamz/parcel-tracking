# Task: Build 3 Parcel Tracking App Screens for Latelogic

We need to develop three Android mobile app screens for a
parcel/shipment tracking app for **Latelogic**.

Please use this Figma design as the main reference:

https://www.figma.com/design/L7TzRfXRoikHtS93kOH00H/Untitled?node-id=0-1&t=ZsROVOu8n85OaLwd-1

The Figma screens are only a design reference, **not the exact final
design**. Please **do not copy them pixel-by-pixel**.

The goal is to **improve the UI** and make it look more polished,
premium, and production-ready while keeping the same overall structure
and flow.

------------------------------------------------------------------------

# Screens to Build

## 1. My Shipments Screen

This is the main home screen where users can see their added shipments.

### Include

-   Page title: **My Shipments**
-   Search button
-   Refresh/Sync button
-   Tabs for **Active** and **Archive**
-   Empty state when no shipment is added
-   Package/parcel-related empty state illustration
-   CTA button: **Track Your First Shipment**
-   Bottom navigation with:
    -   Home
    -   Settings

### Notes

Improve:

-   Spacing
-   Visual hierarchy
-   Empty state design
-   Overall polish compared to the reference

## 2. Add Shipment Screen

-   Courier provider dropdown

-   Shipment name input
-   Push notification toggle card
-   Primary CTA button: **Add Shipment**

## 3. Track Package Details Screen

This screen shows the current shipment status and tracking history.

### Include

-   Title: **Track Package**

    -   Courier logo placeholder
    -   Shipment name
    -   Tracking number
    -   Current status badge
    -   Expected delivery date/time
-   Tracking history section
-   Refresh action
-   Timeline UI showing shipment events:

### Notes

# Design Direction

Use the Figma screens as a **reference only**, not the exact design.

Maintain the overall direction:

-   Clean modern Android mobile UI
-   Light/white background
-   Orange as the primary brand/action color
-   Soft rounded cards
-   Simple icons
-   Clear typography
-   Good spacing and alignment
-   Android-friendly layout and interaction patterns
-   Should feel natural on Android devices, not like a direct iOS copy

The final screens should look **better and more refined** than the
current Figma reference.
